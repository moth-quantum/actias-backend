# Q1Synth v2.0

This app was built with Svelte Kit.

## Local development
* npm install
* npm run dev
* npm run build

## Hosting
The app is currently hosted on Pete Thomas's Netlify account, at the domain:

https://q1synthv2.cephasteom.co.uk/

## Ensemble mode

Copy `src/config-example.js` to `src/config.js`. Fill out the details.

* npm run build
* npm run ensemble
